# Parasiter
## 描述
Parasiter是一个基于React + Typescript + SWC的前端项目，是一个用于医学生学习镜下形态学习的网站
## 项目结构
```
├── src
│   ├── Views : 页面
│   │   ├── Components : 组件
│   │   ├── Assets : 资源文件
│   │   ├── Stores : 状态管理

```
## 项目运行
```
pnpm install
pnpm start
```
## 项目部署
```
pnpm run build
```
## 项目依赖
```
Zustand
React
React-router-dom
NextUI
Tailwindcss
```
